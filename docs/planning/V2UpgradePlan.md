# WebSecScope v2 Upgrade Plan

## 1. Goal
WebSecScope를 단순 스캐너에서 실제 서비스 개선 과정을 기록할 수 있는    
보안 점검 자동화 및 리포팅 도구로 확장

## 2. Core Features
- OWASP Top 10 일부 점검
- SSL/TLS 설정 점검
- 인증/세션 보안 점검
- Linux CIS 일부 점검
- Docker CIS 일부 점검
- Security Score v2
- Before/After 비교 리포트
- HTML Report UI 개선
- 결과 시각화
- LLM 기반 리포트 문장 생성

## 3. Validation Target
- VitaCore
- PhysioSim(Local)

## 4. Portfolio Output
- 1차 점검 결과
- 개선 커밋
- 2차 점검 결과
- Security Score 변화
- 위험도 변화
- HTML Report 캡처
